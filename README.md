## Logo-Hinweis

Das WM-2026-Emblem im Hintergrund basiert auf der von Wikidasher erstellten
[CC-BY-SA-4.0-Version auf Wikimedia Commons](https://commons.wikimedia.org/wiki/File:2026_FIFA_World_Cup_emblem.svg).
Die Darstellung wird für das Hintergrundmuster einfarbig und mit seitlichem Abstand verwendet. FIFA und das
WM-Emblem sind geschützte Marken; dieses Projekt ist eine nichtkommerzielle Uni-Arbeit.
